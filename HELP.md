# Introduction

This API known as the 'Library API' provides RESTful endpoints to manage books in the library.

# Getting Started
## Prerequisites
- Maven 3+
- JDK 8+

### To Build with Tests
    mvn clean package
### To Run the Project Locally
    mvn spring-boot:run
    
##Swagger Documentation
    http://localhost:8080/swagger-ui/index.html
